# Retro FPS Kit - 2.5D Modern Sector Based Game Engine

> **Современный 2.5D движок в стиле шутеров 90-х.**
> Секторальная геометрия (Doom-like), физический рендеринг на OpenGL 4.1 и полный низкоуровневый контроль над графическим пайплайном.

Этот инструмент - основа для создания быстрых, атмосферных шутеров с современной графикой и классической логикой построения уровней.

## 🚀 Основные особенности (Roadmap / Реализовано)

- **Графическое ядро:**
  - **OpenGL 4.1** с использованием **go-gl/gl** для максимального контроля над состоянием видеокарты.
  - Современный рендеринг
  - Полный контроль над визуальным видом сцены
- **Геймплейная архитектура:**
  - **Секторальный движок** (2.5D): геометрия уровней строится на основе секторов (как в Doom/Duke Nukem), что обеспечивает высокую производительность.
  - Физика и коллизии на базе BSP.
- **Низкоуровневая оптимизация:**
  - Минимизация аллокаций в рантайме.
  - Прямое управление буферами (VAO/VBO) и состоянием рендера.

## 🛠 Стек технологий

- **Язык:** Go (Golang)
- **Графика:** OpenGL 4.1 (core profile)
- **Библиотеки:**
  - `github.com/go-gl/gl/v4.1-core/gl` — привязки OpenGL
  - `github.com/go-gl/glfw/v3.3/glfw` — создание окна, ввод
  - `github.com/AllenDang/cimgui-go` — ImGui (инструменты разработчика)
  - `github.com/go-gl/mathgl/mgl64` — математика (матрицы, векторы)

## 📂 Структура проекта

```
.
├── src/
│   ├── main.go/    # Точка входа
│   ├── app/        # Связующее звено
│   ├── core/       # Автономный пакет, никаких импортов извне
│   ├── render/     # Всё что касается отображения
│   ├── scene/      # Формализация сцен (GUI, уровень)
│   └── states/     # Игровые состояния (главное меню, редактор, игровой уровень)
│
└── assets/         # Текстуры, модели, блюпринты
```

Фактически вся суть разработки на **RFK** сводится к написанию своих состояний. Приложение предоставляет **AppAPI**, как интерфейс взаимодействия с базовым функционалом.

## 🔧 Установка и сборка

### Требования

- **Go** 1.21 или выше
- **OpenGL 4.3** совместимая видеокарта
- **GLFW** (системная зависимость)
  - _Ubuntu/Debian:_ `sudo apt-get install libglfw3-dev libgl1-mesa-dev`
  - _Windows:_ обычно поставляется с MinGW или через MSYS2
  - _macOS:_ (внимание: OpenGL 4.3 поддерживается не на всех версиях macOS, рекомендуется Windows/Linux для разработки)

### 🛠️ Сборка

```bash
git clone https://github.com/illoprin/retro-fps-kit-go.git
cd retro-fps-kit-go
go mod tidy
go build -ldflags="-H windowsgui -s -w" -o rfkit.exe .\src\main.go
```

-H windowsgui - создает GUI-приложение
-s - удаляет таблицу символов
-w - удаляет отладочную информацию DWARF
