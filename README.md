**Олдскульная эстетика запечённая в шейдерах**

**RFK** — это возвращение к истокам: секториальная геометрия (Doom-style), честный OpenGL 4.1 и полный низкоуровневый контроль. Никакого мусора.

## 🚀 Философия и Архитектура

Движку насрать, что рендерить. Это чистый хаос, упорядоченный моими интерфейсами. Проект разделен на **Kernel** и **Toolkit**.

### **KERNEL**: Ядро и управление ресурсами

Низкоуровневый оркестратор ресурсов GPU. Здесь живет математика и детерминизм.

- **Deferred Rendering:** Современный пайплайн постобработки «из коробки».
- **RHI (Render Hardware Interface):** Полный контроль над FBO, VBO и выделением VRAM. Никаких скрытых слоев.
- **HDR эффекты:** Bloom, SSAO, Eye Adpaption, Tone Mapping, Color Grading. Словом, база для симпатичной графики
- **App API:** Мост между железом и игрой. Предоставляет системные услуги активному состоянию, не владея ими напрямую.

### **TOOLKIT**: Инструментарий Создателя

Высокоуровневая реализация логики. Использует ESC (Entity Component System), избегая сложностей ООП.

- **Sector-based Logic:** Стены, сектора, коллизии — классика 90-х.
- **Pathfinding & Raycasting:** Быстро, как молния, без лишних абстракций.
- **NPC & Weapons:** Шаблоны для сущностей. Я даю вам глину - лепите из неё демонов.

> Если вам не нравится мой путь — **удалите пакет `toolkit`** и напишите свой. RFK это переварит.

## 🛠 Стек технологий

- **Язык:** Go (Golang) — передача структур через указатели везде.
- **Графика:** OpenGL 4.1 (Core Profile).
- **Математика:** `mgl32`.
- **UI:** `cimgui-go` — потому что инструменты разработчика должны быть удобными.

```text

├── cmd
│   └── rfk
│          rfk.go           # Точка входа

├── pkg/

│   │   └── render/         # Обёртки над GPU ресурсами
│   │
│   └── toolkit/            # Базовый набор инструментов

│       ├── systems/        # Системы (e.g. LevelSystem, LevelRenderer)
│       ├── entities/       # Сущности (e.g. Player, NPC, GUI Text, Particle)
│       └── states/         # Реализации AppState
│
└── assets/                 # Текстуры, модели, шейдеры, блюпринты
```

## 🔧 Установка и Сборка

### Требования

- **Go 1.25+** - наименьший "налог" на cgo.
- **OpenGL 4.1** совместимая видеокарта.

### 🛠️ Сборка

```bash
git clone https://github.com/illoprin/retro-fps-kit-go.git
cd retro-fps-kit-go
go mod tidy
# Сборка чистого бинарника без отладочного мусора:
go build -ldflags="-H windowsgui -s -w" -o rfk.exe .\cmd\rfk\rfk.go
```

`-H windowsgui` - создает GUI-приложение

`-s` - удаляет таблицу символов

`-w` - удаляет отладочную информацию DWARF

## Примечание

> Вся суть разработки на RFK сводится к написанию своих **AppStates**. Приложение дает вам **AppAPI** — используйте его с умом.
